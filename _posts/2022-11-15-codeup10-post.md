## 1411 : 빠진 카드 <br>

```C
#include<stdio.h>

int main() 
{
	int card[50] = {0, }; // 배열 선언
	int n, num;
	
	scanf("%d", &n); // 카드의 총 개수
    for (int i = 1; i <= n - 1; i++) { // 카드 값
        scanf("%d", &num);
        card[num] = 1; // flag 사용해서 카드가 있다면 그 칸의 값을 1로 설정
    }

	for (int i = 1; i <= n; i++) { 
		if (card[i] == 0) // flag가 0이면
			printf("%d", i); // i값 출력
	}
	
    return 0;
}
```

### 느낀점 <br>
* 배열을 이중적으로 탐색하지 않고 1학년 때 잠깐 배웠던 flag의 성질을 이용해보았다.
